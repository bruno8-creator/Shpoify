# Shpoify

Theme Shopify para una tienda centrada únicamente en posavasos.

## Qué incluye

- Home editable con hero, beneficios, selección de estilos, producto destacado, bloque editorial y newsletter.
- Plantillas para producto, colección, carrito, páginas, búsqueda y 404.
- Textos base en español.
- Colores globales editables desde el editor de Shopify.

## Siguiente paso

Conecta este repositorio desde Shopify o sube el theme con Shopify CLI. Después puedes editar imágenes, menús, colecciones y textos desde el personalizador de temas.
